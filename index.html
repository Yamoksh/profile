To accomplish this task, I utilized advanced image processing techniques and implemented computer vision algorithms to detect the iris and pupil in real-time. The algorithm was highly precise and efficient and could be deployed across multiple platforms with ease. This project was a testament to my skillset in software development, problem-solving, and working with sophisticated tools such as Python and OpenCV. The resulting solution has numerous practical applications in fields requiring real-time eye-tracking.
